# Graph Prediction Using Netseer

This repository contains the code for both the R and Python implementations of the `netseer` package.

- The `netseer-r` folder contains the source code for the R implementation
- The `netseer-py` foldr contains the source code for the Python implementation
- The `paper` folder contains the JOSS paper sources

## Paper

The markdown files and assets for the JOSS paper can be found here: [./paper/]

### PDF Generation

The PDF for the JOSS paper is auto-generated via a GitHub Action when changes are detected in the `./paper/` directory.  
The PDF can be found: Under `Actions` in the top tool bar > Clicking on the latest Workflow name > Clicking on `paper` under `Artifacts`.  
This will download a ZIP file with the PDF inside.
