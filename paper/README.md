This folder contains a Journal of Open Source Software (JOSS) submission for _Netseer_.

The source code for _Netseer_ is available at:

* [`../netseer-r`](../netseer-r/) contains the R implementation
* [`../netseer-py`](../netseer-py/) contains the Python implementation


### PDF Generation

For collaborative editing purposes,
the PDF for the JOSS paper is auto-generated via a GitHub Action when changes are detected in this folder.  

The generated PDF can be downloaded as follows:
* In top tool bar within the GitHub interface, click on `Actions`
* Click on the latest Workflow name
* Click on `paper` under `Artifacts`; this will download a ZIP file with the PDF inside
